PROJECT TITLE: Bad Dopo Cream
PURPOSE OF PROJECT: The goal of DOPO's final project will be to create a version of Bad Ice-Cream, which we will call Bad DOPO Cream. This new version of the game will add new features that improve the player experience, including:
• Levels with customizable maps
• Variety of enemies
• Variety of fruits
• Variety of obstacles
